## Deployment

To host Ambient on your server run the following commands:

```
git clone https://github.com/xnoctra/Ambient.git
```
```
cd Ambient
```
```
npm i
```
```
npm start
```

### Updating

```
git pull --force --allow-unrelated-histories
```
